
# AssociationRulesExplorer

A Python app for exploring association rules using an interactive Streamlit tool.

## Features
- Interactive web app for association rule mining
- Supports data upload and exploration
- Visualizes results and provides easy filtering

## Installation

Clone the repository:
```bash
git clone https://github.com/Priya0841/AssociationRulesExplorer.git
cd AssociationRulesExplorer
```

Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app:
```bash
streamlit run interactive_tool.py
```

## Files

| File                | Description                          |
|---------------------|--------------------------------------|
| dataset.csv         | Example dataset for testing          |
| interactive_tool.py | Main Streamlit app                   |
| requirements.txt    | List of Python dependencies          |
